PORTFOLIO

A bootstrap-based résumé and portfolio.

Starting from a base template (see credits in the footer) I added a portfolio section and customized the homepage to create my personal portfolio. I put the focus on accessibility and best practices during the development, to achieve a solid base that can be easily updated in the future. The whole website is developed with Bootstrap 4 and JQuery (dependency of bootstrap). Given its multipage nature, I preferred to use a classical approach to the development, with a simple stack, instead of using React, Angular or Vue. There is no back-end; the whole project is hosted on github pages.
