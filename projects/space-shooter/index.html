SPACE SHOOTER

A simple 2D game built with Unity.

In the Spring 2020, I took part in a Udemy class about Game Development with Unity by Jonathan Weinberger a certified Udemy instructor.
This course taught me how to use Unity and its interface, and how to analyze and solve common Game Development challenges.

Space Shooter is the first project, built to learn the Unity interface and the basic techniques.

It is possible to play the game directly in the browser, thanks to the magic of WebGL; however, it does not yet work on mobile devices.
